Crie um site HTML de script java script com três telas. Todas as telas precisam ter a imagem "logo.png" no cabeçalho. A primeira tela é a "index.html" que possui campos para nome e senha, o botão "Entrar" e link "Esqueci minha senha". Deve ter um código java script para que valide se os campos nome e senha estão preenchidos, para redirecionar para a página "lista_de_produtos.html"  ao clicar no botão "Entrar"  e o link "Esqueci minha senha" redireciona para a página "esqueci_minha_senha.html". A segunda tela é a tela "lista_de_produtos.html". Aqui terá uma lista com dez produtos. Cada produto terá listado o ID do produto, o nome do produto, o preço médio do produto e uma foto do produto. O ID do produto é incremental, os produtos listados são respectivamente: Feijão, Arroz, Macarrão, Farinha de trigo, Fécula de Mandioca, sal, açúcar, café, Dúzia de ovos e leite em pó. Os preços em reais são respectivamente: 6, 5, 4, 7, 10, 1, 3, 3, 20 e 7. As fotos estarão dentro do diretório "fotos" e são respectivamente: feijão.jpeg, arroz.jpeg, macarrão.jpeg, farinha.jpeg, fecula.jpeg, sal.jpeg, açúcar.jpeg, cafe.jpeg, ovos.jpeg e leite.jpeg. A terceira tela é "Esqueci minha senha", que possui o campo "Informar o e-mail cadastrado" e um botão "Redefinir senha". Deve possuir uma função java script para que ao clicar no botão "Redefinir senha", é necessário validar se o campo  "Informar o e-mail cadastrado" está preenchido com algum e-mail. Se validado, apresentar pop-up informando que "O link de redefinição de senha foi enviado para o e-mail informado" e logo após, redirecionar para a tela "Login".
