Crie um site HTML de script java script com três telas. Todas as telas precisam ter a imagem "Logo da empresa" no cabeçalho. A primeira tela é a "Login" que possui campos para nome e senha, o botão "Entrar" e link para "Esqueci minha senha". Os campos nome e senha devem estar obrigatoriamente preenchidos, o botão "Entrar" redireciona para a página "Lista de Produtos" e o link "Esqueci minha senha" redireciona para a página "Esqueci minha senha". A segunda tela é a tela "Lista de Produtos". Aqui, o chatgpt deve pesquisar e os dez produtos de supermercado mais consumidos pelo brasileiro, contendo o ID do produto, o nome do produto, o preço médio do produto e uma foto do produto. A terceira tela é "Esqueci minha senha", que possui o campo "Informar o e-mail cadastrado" e um botão "Redefinir senha". Ao clicar no botão "Redefinir senha", é necessário validar se o campo  "Informar o e-mail cadastrado" está preenchido com algum e-mail. Se validado, apresentar pop-up informando que "O link de redefinição de senha foi enviado para o e-mail informado" e logo após, redirecionar para a tela "Login".
